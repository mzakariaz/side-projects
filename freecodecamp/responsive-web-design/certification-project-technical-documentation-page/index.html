Variables

Here are some examples of variables in Python:

  • Floats: A float is a data type representing a real number. For example, 0.1 and 7.3333333333 are floats in Python. Variables can store the values of floats.
  • Integers: An integer is a data type representing a whole number. For example, 1, 18, 433 are integers in Python.
